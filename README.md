# NumPy Essentials for AI & Data Science

This repository contains practical implementations and exercises using the **NumPy** library. As a **B.Tech CSE (AI)** student, I utilize NumPy as the foundational layer for all numerical computing, matrix operations, and data preprocessing tasks.

## 🚀 Overview

NumPy (Numerical Python) is the core library for scientific computing in Python. This repository documents my mastery of:
* **N-Dimensional Arrays:** Efficient creation and manipulation of `ndarrays`.
* **Vectorization:** Leveraging C-optimized operations to eliminate slow Python loops.
* **Broadcasting:** Handling operations between arrays of different shapes.
* **Linear Algebra:** Matrix multiplication, transpositions, and inversions for AI models.

## 📁 Repository Structure

* `1. Array Foundations.py`: Creating arrays using `np.array`, `np.zeros`, `np.ones`, and `np.linspace`.
* `2. Slicing & Indexing.py`: Advanced techniques for accessing and modifying specific data segments.
* `3. Vectorized Math.py`: Fast mathematical operations across entire datasets.
* `4. Reshaping & Transposing.py`: Preparing data dimensions for Machine Learning model input.
